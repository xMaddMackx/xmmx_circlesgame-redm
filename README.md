# Circles Game Features: 
- 3 Levels: easy, medium, hard.
- 3 Modes: "letters", "numbers", and "mixed".
- (Numbers before letters in the "mixed" mode.)
- Timed in seconds.
- Title: Option to set the time.
- returns `true` for Success and `false` for Failed.
